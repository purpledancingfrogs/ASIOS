# ASIOS Init Agent

This is the minimal executable proof for the ASIOS cognitive agent bootstrap. It creates a structured mission payload and deterministically executes a filesystem operation.

## Files

- `run_asios_agent.py`: Generates and executes a sample mission
- `requirements.txt`: Dependency list (empty for now)
- `hello_from_asios.txt`: Output file created by the mission
- `mission_hello.json`: JSON representation of the deterministic payload

## Usage

```bash
python run_asios_agent.py
